# 贊助金流評估報告（臺灣個人開發者）

> **適用對象**：臺灣居民、個人身份（非公司行號）經營開源工具並接受境外贊助
> ⚠️ **免責聲明**：本文件僅供參考，非法律或稅務建議。實際稅務處理請諮詢專業會計師或稅務師。
> **最後更新**：2026-03-13

---

## 一、平台可用性與提領管道

### 主要贊助平台對臺灣的支援

| 平台 | 臺灣可用 | 提領管道 | 提領幣別 | 最低提領 | 手續費 |
|------|---------|---------|---------|---------|--------|
| **Ko-fi** | ✅ | Stripe / PayPal | USD | $25（Stripe）| Stripe 費率 |
| **GitHub Sponsors** | ✅ | Stripe Connect | USD → 入帳 TWD 或 USD | $100 | 0%（GitHub 不抽成）|
| **Buy Me a Coffee** | ✅ | Stripe / PayPal | USD | $5 | 5% + Stripe/PayPal 費率 |
| **Patreon** | ✅ | PayPal / Payoneer | USD | $25 | 8–12%（依方案）|
| **LemonSqueezy** | ✅ | Stripe | USD | $50 | 5% + $0.5 |
| **OpenCollective** | ✅ | 銀行電匯 | USD | $10 | 10%（Host 費）|

## 二、提領管道詳細評估

### 2-1. Stripe（推薦）

> GitHub Sponsors、Ko-fi、Buy Me a Coffee 均支援 Stripe Connect。

**優點：**
- 可直接綁定**臺灣本地銀行帳戶**（玉山、國泰、台新等皆支援）
- 自動換匯，USD → NTD 直接入帳
- 每次提領費用：**免費**（Stripe 不收提領費，僅收交易手續費 2.9% + $0.30）
- 每月自動撥款，或達閾值立即撥款

**流程：**
```
贊助者付款（USD）
  → Stripe 扣除交易費（2.9% + $0.30/筆）
  → 帳戶餘額（USD）
  → 自動換匯 + 匯入臺灣銀行帳戶（NTD）
  → 通常 2–3 個工作日到帳
```

**換匯說明：**
- Stripe 使用市場匯率，通常比銀行牌告匯率略低 0.5–1%
- 無額外換匯手續費

---

### 2-2. PayPal

**優點：**
- 普及率高，許多贊助者習慣使用

**缺點：**
- 提領至臺灣銀行帳戶：每筆收 **$0.50 USD 或等值固定費**
- PayPal 自訂匯率通常比市場匯率差 2.5–3.5%
- 臺灣 PayPal 帳戶功能有限制（無法直接接受信用卡付款，僅 P2P）
- 近年在臺爭議較多（帳號凍結案例）

**建議：** 避免作為主要提領管道，若使用 PayPal 收款，考慮再轉至 Wise 後提領。

---

### 2-3. Wise（前 TransferWise）

> 不直接整合贊助平台，但可作為**中間換匯帳戶**。

- 開設美元虛擬帳戶，接收來自 PayPal / 平台的 USD
- 換匯費率接近市場中間價，費率 0.4–0.6%（遠優於銀行）
- 提領至臺灣銀行：每筆約 NT$30–50 固定費
- **適合情境**：從 PayPal 或 Patreon 收款後，透過 Wise 換匯再入臺帳戶

---

### 2-4. Payoneer

- 支援臺灣，可綁定臺灣銀行
- 提領至本地帳戶：每筆 $3 USD 固定費（年費 $0）
- 匯率：較市場差 2%
- 部分平台（如 Patreon）支援 Payoneer 直連

## 三、稅務評估

> ⚠️ 以下為 **個人、非公司行號** 的一般情況說明，實際情形請諮詢會計師。

---

### 3-1. 所得稅：境外所得

**法規依據：** 《所得基本稅額條例》（最低稅負制）

| 項目 | 說明 |
|------|------|
| 性質 | 來自境外平台（Ko-fi、GitHub 等）的贊助款，屬**境外所得** |
| 免稅門檻 | 境外所得全年未超過 **新臺幣 100 萬元**，可免計入最低稅負 |
| 超過門檻 | 須申報於「個人最低稅負制」（基本所得額），稅率 20% |
| 年度申報時間 | 每年 5 月（結算申報） |

**實際試算：**

```
假設年贊助收入 $500 USD ≈ NT$16,000
→ 遠低於 NT$100 萬門檻
→ 實務上無需申報境外所得，稅負為 $0
```

```
假設年贊助收入 $10,000 USD ≈ NT$320,000
→ 仍低於 NT$100 萬門檻
→ 無需申報境外所得
```

```
假設年贊助收入 $35,000 USD ≈ NT$1,120,000
→ 超過 NT$100 萬門檻
→ 需申報，計入最低稅負基本所得額
→ 基本所得額超過免稅額 NT$670 萬方需繳稅
（一般個人開源工具，觸及此門檻機率極低）
```

---

### 3-2. 境內所得認定問題

若贊助平台將款項認定為**服務費**（而非捐款），有可能被視為「執行業務所得」或「其他所得」：

| 認定方式 | 所得類別 | 稅率 |
|---------|---------|------|
| 捐款 / 贊助（無對價）| 通常不列為所得（灰色地帶）| — |
| 提供軟體使用授權 | 權利金所得 | 併入綜所稅 |
| 提供技術服務 | 執行業務所得 | 併入綜所稅 |
| 其他收入 | 其他所得 | 併入綜所稅 |

**實務建議：**
- 以「贊助」（donation）定位，而非「銷售」或「訂閱服務」，降低稅務複雜度
- 若有「對價提供功能」，性質趨近服務費，需更謹慎處理

---

### 3-3. 營業稅（VAT）

| 門檻 | 說明 |
|------|------|
| 年銷售額未達 **NT$48 萬** | 免辦稅籍登記（小規模營業人豁免）|
| 年銷售額 NT$48 萬 ~ NT$240 萬 | 小規模營業人，查定課徵 1% |
| 年銷售額超過 NT$240 萬 | 一般營業人，2% 或 5% 稅率 |

**評估：**
- 個人接受「贊助」（無實體商品交易）通常**不構成營業行為**
- 若以「解鎖碼換取功能」方式收費，性質接近軟體銷售，可能需要考慮
- 小額贊助（年收入幾千到幾萬台幣）不太可能觸及門檻

---

### 3-4. 扣繳問題（境外平台的 W-8BEN）

> 美國平台（GitHub、Ko-fi 等）可能要求填寫 **W-8BEN 表格**，聲明為非美國人，以避免美方扣繳 30% 預扣稅（withholding tax）。

**臺灣與美國有稅務協定（有限度）：**

| 情形 | 說明 |
|------|------|
| 填寫 W-8BEN | 聲明臺灣居民身份，適用協議稅率或豁免 |
| 未填寫 | 美方可能自動扣繳 30% |
| GitHub Sponsors | 要求填寫 W-8BEN，填後通常不扣繳 |
| Ko-fi / Stripe | 通常不涉及扣繳（P2P 捐款性質）|

**建議：** 所有美國平台均填寫 W-8BEN，避免被扣 30%。

## 五、實務建議路徑

```
[初期] 年收入 $0 ~ $500 USD
  → Ko-fi 榮譽模型（不設解鎖碼）
  → Stripe 直接入臺灣銀行帳戶
  → 稅務：無需特別處理

[成長期] 年收入 $500 ~ $5,000 USD
  → GitHub Sponsors（手續費最低）
  → 填寫 W-8BEN（向 GitHub 聲明臺灣居民）
  → 稅務：境外所得仍低於 NT$100 萬門檻，無需申報
  → 建議：保存完整收款紀錄備查

[規模化] 年收入 $5,000+ USD
  → 評估是否以公司或工作室名義登記（稅務更清晰）
  → 諮詢會計師確認所得性質與申報方式
  → 考慮加入 Open Collective（有正式會計紀錄）
```

## 七、常見 Q&A

**Q：贊助算「收入」還是「捐款」，一定要申報嗎？**
A：臺灣所得稅對「境外贊助款」無明確規定，個人少量贊助實務上多不申報。若金額較大，建議諮詢會計師，以「其他所得」評估。

**Q：需要開發票給贊助者嗎？**
A：個人身份、境外贊助者，通常不需開立臺灣統一發票。部分平台（如 Ko-fi）有自己的交易記錄可供贊助者自行取得。

**Q：Stripe 入帳的錢，銀行會通報嗎？**
A：臺灣銀行對**單筆超過 50 萬台幣**的匯入有申報義務（金融機構反洗錢規定），小額贊助通常不觸及。若長期每月有固定入帳，銀行通常不會主動詢問，但建議保留收款來源說明文件。

**Q：GitHub Sponsors 的 W-8BEN 怎麼填？**
A：填寫個人基本資料（姓名、地址、國籍：Taiwan）和 Tax Identification Number（可填臺灣身份證字號）。選擇「個人」類型，聲明非美國人即可。
